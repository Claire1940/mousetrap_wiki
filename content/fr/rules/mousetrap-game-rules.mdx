<YouTubeEmbed videoId="RJPOHG6ing8" title="Comment jouer à Attrap'Souris" />

Donnez à chaque joueur une part de fromage pour commencer la partie. Placez deux parts de fromage bonus directement sur la case Départ. Les parts de fromage restantes doivent être placées à côté du plateau pour former la pile de fromage principale. Choisissez un joueur pour commencer, et le jeu se poursuivra dans le sens des aiguilles d'une montre.

| Composant | Quantité | Objectif |
| :--- | :--- | :--- |
| **Pions Souris** | 4 | Représentent les joueurs sur le plateau |
| **Parts de Fromage** | Plusieurs | Collectées pour gagner (besoin de **six parts** pour gagner) |
| **Dé de Jeu** | 1 | Détermine la distance de déplacement |
| **Pièces du Piège** | 1 Ensemble | Mécanisme assemblé pour capturer les souris adverses |

<Callout type="warning" title="Conseil d'installation important">
Assurez-vous que le mécanisme physique du piège est entièrement assemblé et testé avant de commencer la partie pour éviter les défaillances mécaniques en cours de jeu.
</Callout>

Tous les joueurs doivent commencer sur la case Départ désignée. Lorsqu'un joueur termine son premier tour de plateau et passe par la case Départ, il collecte immédiatement les deux parts de fromage bonus qui y ont été placées au début de l'installation.

| Case | Action | Résultat |
| :--- | :--- | :--- |
| **Départ** | Position de départ | **Deux parts de fromage bonus** pour le premier joueur à terminer un tour |
| **Jaune** | Case Fromage | Collectez **une part de fromage** dans la pile principale |
| **Vert** | Case Vol | Prenez **une part de fromage** à n'importe quel adversaire |
| **Rouge** | Case Perte | Rendez **une part de fromage** à la pile centrale |
| **Bleu** | Case Lancement | Tentez de piéger un adversaire sur la **case piège** |
| **Violet** | Case Vide | Aucune action ne se produit ; le tour se termine |
| **Tunnel** | Case Souricière | **Déplacement optionnel** directement à l'autre extrémité du tunnel |

À votre tour, lancez le dé et déplacez votre pion exactement de ce nombre de cases dans le sens des aiguilles d'une montre. Selon la case sur laquelle vous tombez, vous devez résoudre immédiatement des actions spécifiques. Le plateau comporte plusieurs types de cases qui peuvent vous aider à accumuler du fromage ou vous faire perdre votre progression.

<Callout type="tip" title="Mouvement Stratégique">
Utilisez le tunnel de la souricière de manière stratégique pour vous positionner plus près des cases de fromage ou pour éviter les adversaires qui sont prêts à vous piéger.
</Callout>

Si vous tombez sur une case fromage jaune, collectez une part de fromage dans la pile principale. Tomber sur une case vol verte vous permet de prendre une part de fromage à n'importe quel autre joueur de votre choix. Si vous tombez sur une case perte rouge, vous devez rendre l'une de vos parts de fromage collectées à la pile principale. Tomber sur une case violette n'entraîne aucune action et votre tour se termine immédiatement.

<Section icon={Swords} title="Comment lancer le piège et attraper des souris">

La mécanique interactive centrale du jeu consiste à déclencher le piège physique pour capturer les souris adverses. Cette action est déclenchée lorsqu'un joueur tombe sur la case de lancement bleue. Déclencher le piège nécessite une exécution soignée et un alignement des composants mécaniques.

<Steps>
  <Step num="1" title="Sélectionner la Cible">
    Choisissez un joueur et déplacez son pion directement sur la case piège désignée.
  </Step>
  <Step num="2" title="Déclencher le Mécanisme">
    Tirez sur le composant du panneau stop et relâchez-le pour démarrer la réaction en chaîne.
  </Step>
  <Step num="3" title="Résoudre le Résultat">
    Si le piège capture avec succès le joueur, celui-ci doit vous donner l'une de ses parts de fromage.
  </Step>
  <Step num="4" title="Réinitialiser le Piège">
    Remettez la bille, le plongeur, la cage et le balai dans leurs positions de départ d'origine.
  </Step>
</Steps>

Si le piège ne parvient pas à capturer la souris cible pour une raison quelconque, le joueur ne vous donne pas de fromage. S'il y a déjà une souris sur la case piège lorsque vous tombez sur la case de lancement bleue, vous pouvez tout de même y envoyer une autre souris. Si le piège réussit dans ce scénario, vous collectez une part de fromage de tous les joueurs piégés simultanément.

| Scénario | État du Piège | Résultat |
| :--- | :--- | :--- |
| **Une seule souris capturée** | Succès | Le joueur ciblé vous donne **une part de fromage** |
| **Plusieurs souris capturées** | Succès | Tous les joueurs piégés vous donnent **une part de fromage** chacun |
| **Le piège échoue/rate** | Échec | Aucun fromage n'est échangé ; réinitialisez le piège |
| **Aucun adversaire envoyé** | N/A | Impossible de déclencher le piège sans cible |

<Callout type="success" title="Conseil pour réussir le piège">
Assurez-vous toujours que le balai est tourné vers le haut et que la bille repose solidement dans le seau du plongeur avant de lancer pour maximiser la fiabilité du piège.
</Callout>

Une fois l'action du piège résolue, le joueur actif doit réinitialiser tous les composants. Cela inclut le retour de la bille, du plongeur et de la cage à leurs positions d'origine, garantissant que le plateau est prêt pour le tour du joueur suivant.

</Section>

<Section icon={Trophy} title="Stratégies gagnantes et liste de contrôle">

Gagner la partie nécessite un équilibre entre le piégeage agressif et la collecte régulière de fromage. Bien que déclencher le piège soit un moyen puissant de voler du fromage, compter uniquement sur les cases de lancement peut être risqué si les composants mécaniques ne se déclenchent pas correctement.

<Callout type="info" title="Note stratégique">
Bien que le piégeage soit excitant, tomber sur les cases vertes et jaunes est souvent le moyen le plus constant de sécuriser des parts de fromage.
</Callout>

| Style de jeu | Priorité | Avantage principal | Niveau de risque |
| :--- | :--- | :--- | :--- |
| **Piégeur Agressif** | Cases bleues et piégeage | Récompense élevée contre plusieurs adversaires | **Élevé** (dépend de la mécanique du piège) |
| **Collecteur Passif** | Cases jaunes et vertes | Revenu de fromage régulier et fiable | **Faible** (moins dépendant du succès physique du piège) |

Les joueurs doivent surveiller de près le nombre de fromages de leurs adversaires. Si un adversaire est sur le point de collecter six parts, ciblez-le exclusivement avec des cases de vol vertes ou envoyez-le sur la case piège pour retarder sa victoire.
